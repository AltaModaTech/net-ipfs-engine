Bootstrap Peers

The bootstrap peers, aka seed nodes, are the multiaddress(es) of well known and trusted peers that have high availability and is used to find other peers on the IPFS network.

These peers are listed in the configuration file and are managed with the Bootstrap API.

The Discovery options can be used to override the default bootstrap peers. When using a private network new bootstap peers must be defined.
