Database

A SQLite database, named ipfs.db, stores most of the repository information. You can use DB Browser to examine and modify the information.

Table Usage Description
Configs Config API The configuration information
EncryptedKeys Key API An encrypted key
Keys Key API Details on a keys
Pins Pin API A Block that is pinned to the local repository
_EFMigrationsHistory Current version of the database

Migration

From time to time the database schema changes. _EFMigrationsHistory records the current version of the database. At initialisation, the database is upgraded/downgraded based on the version of IPFS Engine.
